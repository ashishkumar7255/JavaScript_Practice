// Select all elements with the class "tab1" and store them in the variable t1 (NodeList of all tabs)
var t1 = document.querySelectorAll(".tab1");

// Select the element with id "alen" and store it in the variable aln
var aln = document.querySelector("#alen");

// Select the element with id "steve" and store it in the variable st
var st = document.querySelector("#steve");

// Select the element with id "sunder" and store it in the variable sun
var sun = document.querySelector("#sunder");

// Initially, make the "alen" section visible
aln.style.display = "block";

// Initially, hide the "steve" section
st.style.display = "none";

// Initially, hide the "sunder" section
sun.style.display = "none";

// Loop through each tab in the NodeList t1
t1.forEach((tab, index) => {

// Add a click event listener to each tab
tab.addEventListener("click", function () {

// Hide all sections when a tab is clicked
aln.style.display = "none";
st.style.display = "none";
sun.style.display = "none";

// Check which tab is clicked and show the corresponding section
if (index === 0) {
// If the first tab (index 0) is clicked, show "alen"
aln.style.display = "block";
} else if (index === 1) {
// If the second tab (index 1) is clicked, show "steve"
st.style.display = "block";
} else if (index === 2) {
// If the third tab (index 2) is clicked, show "sunder"
sun.style.display = "block";
}
});
});

Musk developed an interest in technology and entrepreneurship at an early age. He taught himself computer programming and sold his first software, a game called Blastar, at the age of 12. After moving to the United States, he attended the University of Pennsylvania, earning degrees in Physics and Economics. He then briefly joined a Ph.D. program at Stanford University but dropped out after two days to pursue his entrepreneurial dreams.Musk's first major success was Zip2, a web software company that provided business directories and maps to newspapers. He sold Zip2 for nearly $300 million and co-founded X.com, an online payment company that later became PayPal. When eBay acquired PayPal in 2002 for $1.5 billion, Musk received a significant financial boost, allowing him to invest in more ambitious projects.

Steve Jobs was born on February 24, 1955, in San Francisco, California. He was adopted by Paul and Clara Jobs, who nurtured his interest in electronics and mechanics. Jobs attended Reed College but dropped out after six months, preferring to learn through personal exploration. He later traveled to India in search of spiritual enlightenment, an experience that influenced his design philosophy.In 1976, Jobs co-founded Apple with his friend Steve Wozniak in his parents garage. They created the Apple I, a personal computer that gained attention for its innovative design. This was followed by the Apple II, which became one of the first mass-produced personal computers. In 1984, Apple launched the Macintosh, a computer with a graphical user interface and a mouse—revolutionary concepts at the time. However, due to internal conflicts, Jobs was forced to leave Apple in 1985
